# Welcome to Devopsark CLI Installation!

This is the guide for installing devopsark cli on your server !

# Linux

Step 1 : Download Binary from https://github.com/Devopsark/cli/releases

Step 2 : Create a directory anywhere on your machine 

Step 3 : Move the binary in newly created directory.

Step 4 : Rename the binary to `devopsark`

Step 5 : Add that directory to PATH in environment variables

Step 7 : Open Terminal and type `devopsark`

# Windows

Step 1 : Download Binary from https://github.com/Devopsark/cli/releases

Step 2 : Create a directory anywhere on your machine 

Step 3 : Move the binary in newly created directory.

Step 4 : Rename the binary to `devopsark`

Step 5 : Add that directory to PATH in environment variables

Step 6 : Open CMD and type `devopsark`

# MacOS

Step 1 : Download Binary from https://github.com/Devopsark/cli/releases

Step 2 : Create a directory anywhere on your machine 

Step 3 : Move the binary in newly created directory.

Step 4 : Rename the binary to `devopsark`

Step 5 : Add that directory to PATH in environment variables

Step 6 : Trust the binary in "Privacy and Security" (This option will appear after you try to execute it once )

Step 7 : Open Terminal and type `devopsark`

## Notes :
